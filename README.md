"# SchoolSystem"
A list of what you can do:
1. Add a New Student from the keyboard
2. Remove a Student
3. Find a Student by ID
4. Add a class for one student
5. Drop a class for one student
6. Print the list of student names in a class
7. Print out the transcript of a student
8. Show all students
9. Exit

At the school ABC, the information of students is stored in two files:
- the file studentInformaiton includes: student ID, last name, first name, Social Security Number, birth day, phone number
- the file studentGrade include student ID and all the classes the students take/have taken with the letter grade

I used Binary Search Tree as the search algorithm for this project.
